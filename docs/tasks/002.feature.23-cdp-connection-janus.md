# Task: CDP Connection Mode + Janus Integration

## Summary

Add support for connecting to an existing browser via Chrome DevTools Protocol (CDP) instead of always launching a new one. Then add a Janus-specific helper that creates web tabs via Janus's HTTP API (port 9223) and connects to them via CDP (port 9222).

## Motivation

Puppet currently only launches its own Chromium instance. For integration with Janus (Electron app with web panel tabs), external tools need to:

1. Create a web tab inside Janus via its HTTP API
2. Connect to that tab's webview via CDP
3. Automate it with the full Puppet API

This also enables connecting to any running Chrome instance (`--remote-debugging-port`), which is useful beyond Janus.

## Current State

- `browser.ts` only has `launchBrowser()` — always spawns a new Chromium
- `fluent.ts` `Browser` class wraps a `Session` — sessions always launch a new browser
- `session.ts` `startSession()` calls `getBrowser()` which calls `launchBrowser()`
- No CDP connection support anywhere in the codebase
- Abra has its own `connectOverCDP` code in `cli.ts:188-202` but it bypasses Puppet entirely

## Implementation

### Phase 1: CDP Connection in `browser.ts`

Add a `connectCDP` function alongside `launchBrowser`:

```typescript
// browser.ts
export async function connectCDP(cdpUrl: string, options?: BrowserOptions): Promise<BrowserInstance> {
  const { chromium } = await import('playwright');
  const browser = await chromium.connectOverCDP(cdpUrl);
  const contexts = browser.contexts();
  const context = contexts[0] || await browser.newContext({ viewport, userAgent, ... });
  const pages = context.pages();
  const page = pages.find(p => /* match by URL if specified */) || pages[0] || await context.newPage();
  return { browser, context, page, videoEnabled: false };
}
```

### Phase 4: Janus Helper

Add `janus.ts` with a convenience function:

```typescript
// janus.ts
const JANUS_API = 'http://localhost:9223';
const JANUS_CDP = 'http://localhost:9222';

export interface JanusOptions extends Omit<SessionOptions, 'cdp'> {
  /** Janus HTTP API URL. Default: http://localhost:9223 */
  apiUrl?: string;
  /** Janus CDP URL. Default: http://localhost:9222 */
  cdpUrl?: string;
  /** URL to open in new web tab */
  url: string;
}

export async function janusTab(options: JanusOptions): Promise<Browser> {
  const apiUrl = options.apiUrl || JANUS_API;
  const cdpUrl = options.cdpUrl || JANUS_CDP;

  // Create tab via Janus HTTP API
  const res = await fetch(`${apiUrl}/api/tabs`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ url: options.url }),
  });
  const { tabId } = await res.json();

  // Connect via CDP, find the page by URL
  const browser = await puppet({
    ...options,
    cdp: cdpUrl,
    cdpPageUrl: options.url,
  });

  return browser;
}

// Tab management helpers
export async function janusListTabs(apiUrl = JANUS_API) {
  const res = await fetch(`${apiUrl}/api/tabs`);
  return res.json();
}

export async function janusCloseTab(tabId: number, apiUrl = JANUS_API) {
  await fetch(`${apiUrl}/api/tabs/${tabId}`, { method: 'DELETE' });
}

export async function janusNavigateTab(tabId: number, url: string, apiUrl = JANUS_API) {
  await fetch(`${apiUrl}/api/tabs/${tabId}/navigate`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ url }),
  });
}
```

Usage:

```typescript
import { janusTab, janusCloseTab } from 'puppet';

const browser = await janusTab({ url: 'http://localhost:3000' });
await browser.click('login-btn');
await browser.type('email-input', 'test@example.com');
await browser.screenshot();
await browser.close();
```

## Notes

- No video recording when connecting via CDP (Playwright limitation — can only record contexts it creates)
- `close()` on a CDP-connected browser disconnects but doesn't close the browser/tab
- Page matching by URL uses `page.url().includes(cdpPageUrl)` — substring match
- Janus HTTP API is at port 9223, CDP at port 9222 (both hardcoded in Janus `main.js`)
